Read Offline App

This mini website is a demo for HTML5 based web application ReadOfflineApp. The home page is used to put requested files in appcache manifest. The 3 pages HTML CSS WEBFONTS contain some basic articles, just to fill content with something readable, slightly different from lorem ipsum. The last page called "offline" contains the basic html structure for reading stored articles when you're offline. Use navigation menu at the top of the page, to browse the site.
